# ReplayUI

Custom replay user interface for StarCraft II Co-op.

https://www.maguro.one/2019/10/replay-interface.html

# Features
* Ability to hide different UI elements in replay
* Shows masteries for both players
* Shows APM, damage and kills for each player
* Shows various statistics

# Keyboard shortcuts

* CTRL+ H - shows/hides all UI
* CTRL+ I  - shows/hides info panel
* CTRL+ D - shows/hides masteries
* CTRL+ F - shows/hides scores
* CTRL+ R - shows/hides replay UI
* CTRL+ C - shows/hides chat
